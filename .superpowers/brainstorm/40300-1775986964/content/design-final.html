Abadikan Landing — Final Design (Mobile + Desktop)

Post research: linear.app · stripe · tokopedia eng · Chrome DevRel · MDN performance

Temuan terpenting untuk mobile

Tokopedia, Shopee, Gojek tidak pakai parallax atau GSAP sama sekali di mobile web. Premium feel datang dari: tap feedback <80ms, scroll-snap native, reveal 500ms (bukan 850ms), dan translateY max 16px (bukan 28px). Less is genuinely more.

CSS scroll-driven animations

animation-timeline: view() berjalan di compositor thread — lebih smooth dari IntersectionObserver. Support: Chrome 115+ (mayoritas Android). Pakai @supports dengan fallback IO yang sudah ada. Zero JS overhead untuk reveals standar.

Yang harus dihapus dari kode saat ini

Transition duration 850ms → 500ms. translateY 28px → 16px. Stagger max delay 880ms → 400ms. Ini yang bikin site sekarang terasa "berat" di mobile bukan "smooth".

GSAP pin = jangan di mobile

GSAP ScrollTrigger pin: true pecah di Android saat URL bar collapse/expand dan saat keyboard muncul. Gallery horizontal: GSAP hanya desktop ≥1024px. Mobile: CSS scroll-snap native.

Global Fixes — Berlaku di Semua Section

Fix timing reveal (kode sekarang terlalu lambat)

.landing-reveal transition: 500ms ← was 850ms
translateY: 16px ← was 28px
max stagger delay: 400ms ← was 880ms

850ms terasa "berat" di mobile. 500ms + expo-out easing = crisp dan premium.

Tambah ke globals.css

html { overscroll-behavior-y: contain; }
* { -webkit-tap-highlight-color: transparent; }
.btn:active { transform: scale(0.96); transition: 80ms; }

Prevent pull-to-refresh accidental. Remove gray tap flash. Native-feeling button feedback.

CSS scroll-driven animations (Chrome 115+)

@supports (animation-timeline: view()) {
  .landing-reveal { animation: reveal-in ... }
  animation-timeline: view();
}

Fallback ke IO yang sudah ada. Compositor thread = 60fps guaranteed on mid-range Android.

Accordion fix (landing-FAQ)

grid-template-rows: 0fr → 1fr
← replaces max-height transition
transition: grid-template-rows 300ms ease

max-height triggers layout recalc tiap frame (D-tier). grid-template-rows tidak — compositor safe.

Parallax = desktop only

@media (hover: hover) {
  /* parallax effects here */
}

Tokopedia, Shopee, Gojek zero parallax di mobile. Vestibular + performance concern.

Navbar blur performance

.navbar {
  backdrop-filter: blur(10px);
  contain: paint; /* isolates repaint */
}

Max blur(12px). contain: paint mencegah blur mempengaruhi repaint seluruh page.

Final Tech Stack

lenis ~3KB — smooth scroll
gsap ~48KB — desktop ≥1024px only, lazy
animation-timeline: view() — native CSS, 0KB
useScrollReveal.ts — enhanced (timing fix)
IntersectionObserver — kept as @supports fallback
max-height accordion — diganti grid-template-rows
Bundle impact on mobile
Mobile user download: +3KB (Lenis only). GSAP tidak diload sama sekali.
Semua animasi mobile berjalan di compositor thread — 60fps on Snapdragon 600-series.
CSS scroll-snap carousels: zero JS, handled by browser natively.
